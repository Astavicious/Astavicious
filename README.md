# Featured Projects

## 🤖 AI Learning Assistant

An AI-powered learning assistant that answers questions from uploaded PDF documents and generates summaries, quizzes and study material.

**Technologies:** Python, LangChain, Ollama, ChromaDB, FastAPI and Streamlit.

Main features:

- PDF document upload
- Retrieval-Augmented Generation
- Citation-based answers
- Automatic summaries
- Quiz generation
- Locally hosted language models

---

## 🧠 Stroke Risk Prediction

A machine learning project that predicts stroke risk from patient health information.

**Technologies:** Python, Pandas, NumPy, Scikit-learn, Matplotlib and Jupyter Notebook.

Main features:

- Data preprocessing
- Exploratory data analysis
- Model training
- Model evaluation
- Prediction interface

---

## 🌐 Network Simulation Analytics

An automated framework for evaluating congestion-control algorithms using NS-3 simulations.

**Technologies:** Python, NS-3, automated testing and data visualization.

Main features:

- Simulation automation
- Congestion-control comparison
- Throughput analysis
- Delay analysis
- Packet-loss analysis
- Performance dashboard
